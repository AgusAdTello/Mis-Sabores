# Mis Sabores - Conservas Artesanales

Sitio web promocional diseñado para exhibir una línea de mermeladas y dulces artesanales. El enfoque principal es la presentación visual de los productos y la facilitación del contacto con los clientes.

## 📸 Características
- Galería de productos destacados.
- Diseño visualmente atractivo centrado en la marca.
- Sección de contacto directa.
